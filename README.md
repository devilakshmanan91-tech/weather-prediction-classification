# Weather Prediction using Machine Learning (Classification)

## Overview
This project predicts weather conditions using machine learning classification techniques.
The dataset was collected from Kaggle and processed using Python.

## Dataset
- Source: Kaggle
- Features include temperature, humidity, wind speed, and pressure

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Workflow
1. Data collection
2. Data preprocessing
3. Feature engineering
4. Model training
5. Model evaluation

## Results
- The model achieved good accuracy in classifying weather conditions.
- Random Forest showed better performance among the models.
